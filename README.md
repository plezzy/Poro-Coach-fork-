# Poro-Coach
Bot de Discord que, mediante técnicas de web scraping o acceso a APIs públicas, permita consultar y mostrar la información de builds, runas y counters de campeones del juego League of Legends, con comandos específicos (por ejemplo !build &lt;campeón>), de modo que los usuarios obtengan la información directamente en el chat sin salir de la plataforma.
